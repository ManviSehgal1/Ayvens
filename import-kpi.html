Import KPIs

To import the Target values for the selected KPIs:

  1. Select at least one KPI from the table and export the work template by clicking on "Export pre-filled" that will appear above;
  2. Fill in the relevant columns;
  3. Close the file;
  4. Click on the "Import" button
  5. Select the file to import in the selection window that will appear automatically
